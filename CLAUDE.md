# Project: sninfole.github.io — personal academic site
Owner: Stefano Ninfole, PhD researcher in Economics (NMBU).
Stack: Jekyll on GitHub Pages. Content lives in markdown.
RULE: never change my wording — only layout, structure, styling.

Goal: clean, professional academic site, responsive. Editorial monochrome
look ("classy, not sterile"): ink on warm paper, hairline rules, serif display
headings (Fraunces) + sans body (Inter), generous whitespace. Dark-mode variant:
- paper #FAF9F7 (page bg) / dark #141412
- ink #1B1A17 (text, buttons, rules)
- slate #6E6A62 (muted text)
- mist #E5E2DC (hairline borders)
Links are ink-colored with grey underlines. No color accents.

Sections (keep all existing content): hero/about + photo, Research Impact
(narrative, outcome-focused storytelling), Publications, Working Papers
(with descriptions/coauthors/status), Teaching, CV download, Contacts.
Future: a separate page embedding an R-generated interactive map via iframe.
